A bit about my services

Static Websites

These can be personal, marketing or landing pages. I build these from scratch using HTML, CSS and javascript.

Dynamic content sites

These websites are based on a CMS, (content management system) and are most often used for blogs or magazines and are built using open source sytems such as Wordpress.

E-commerce

E-commerce sites are typically built using an online platform and a templating system. They are very similar in structure to Wordpress sites. A template can be developed from scratch or a a premade template can be customised.

Web apps

Web apps may be built on either the front or back end, but every app needs an HTML structure and CSS styling when facing the user. In this way there are the same as any other website. Some programming frameworks also have their own templating language and I am comfortable using these to build your front end.
