A bit about Vielife

Details

Vielife engaged in a complete redesgin of their online health tool for employees. In Spring 2016 I built a responsive static version of this new design using HTML, Bootstrap, CSS3 and jquery which was to be plugged in to their .NET backend. The software includes over 50 separate screens including numerous form elements. Vielife (Cigna) put particular emphasis on accessiblity and so I followed WCA Guidelines closely. Some of the navigation menus were particularly challenging to develop in CSS for both mouse and keyboard users because of "pop out"/visibility functionality that was designed with hovering mouse pointers in mind more than keyboard focus.
